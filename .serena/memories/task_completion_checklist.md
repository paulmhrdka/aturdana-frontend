# Task Completion Checklist

After completing a coding task in this project:

1. **Lint**: `npm run lint` — fix any ESLint errors
2. **Type check**: `npx tsc --noEmit` — ensure no TypeScript errors
3. **Manual test**: `npm run dev` and verify the feature works in browser
4. **Build check** (for significant changes): `npm run build` — ensure production build passes

No automated test suite is present in the project yet.
